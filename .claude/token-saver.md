# Token Saver Rules

- Never read entire files unless explicitly asked
- Read only the specific function/component needed
- Use grep/search to find relevant lines first
- No explanations unless asked
- No "I will now..." or "Let me..." preambles
- No summaries after completing tasks
- Just write the code and stop
- Keep all responses under 200 words
- Never re-read files that were just edited
- Use str_replace for small edits, never rewrite entire files
- Run npm run build only when explicitly asked
